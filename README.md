Boa tarde,

Como base no que foi solicitado, somente não deu tempo de corrigir o código para que seja enviado o formulário preenchido por e-mail, assim algumas outras funções de layout, no entando, ao abir o index.html, irá aparecer mostrar o layout que optei por construir utilizando o conceito de Mobile first e responsividade.

Ao alterar a dimensão do layout para as dimensões solicitadas, é possível notar mudanças no layout do menu e no layout da parte central do webpage.
os botões de "Entre em contato", assim que clicados redirecionam a página para o rodapé onde fica o formulário a ser preenchido.

Os ícones de "facebook", "instagram, "youtube" e "linkedin" foram configurados para redirecionar para a página inicial dos seus respectivos sites, porém podendo ser alterado para a página da empresa em questão.

Quando no layout menor que 768px de largura, o menu é alterado para o estilo "hamburguer" e ao clicar nele, abre o menu em fullscreen mostrando os mesmos ícones do "facebook", "instagram", "youtube" e "linkedin".
